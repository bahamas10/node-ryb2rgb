ryb2rgb
=======

Convert colors in JavaScript from ryb to rgb

Example
-------

### ryb2rgb

``` js
var ryb2rgb = require('ryb2rgb');
console.log(ryb2rgb([0, 255, 0]));
```

yields

``` js
[255, 255, 0]
```

Usage
-----

``` js
var ryb2rgb = require('ryb2rgb');
```

or

``` html
<script src="ryb2rgb.js"></script>
// defines ryb2rgb()
```

Installation
------------

    npm install ryb2rgb
